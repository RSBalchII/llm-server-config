# Help and Commands

## Triggers
- help
- what can you do
- commands
- capabilities

## Pattern
`\b(help|what\s+can\s+(you|u)\s+do|commands|capabilities|features)\b`

## Command
`echo "Commands: list files, show <file>, create <file> with <content>, search <text>, git <cmd>, system/disk/memory/cpu info"`

## Safe
true

## Description
Show available commands
